# 공연 조회 프로그램  

### 프로젝트 목적
- 공공API(공연예술통합전산망API)를 토대로 공연에 대한 정보를 일자별, 제목별 조회

### 내용 : 
- 클라이언트 : 공연 정보조회, 회원가입, 관심있는 공연 저장
- 서버 : 정보 제공

### 기능
- 회원관리 : 회원가입, 비밀번호 수정, 관심 있는 공연 정보 저장
- 공연/ 공연시설 조회 : 일자 별, 제목 별 조회
